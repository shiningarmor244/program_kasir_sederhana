# Program Kasir Sederhana

Program kasir sederhana adalah sebuah program yang dikembangkan untuk mempermudah proses transaksi. Program ini dibuat menggunakan bahasa python dan dirancang untuk menampilkan daftar menu dan harga, menghitung total belanja dengan diskon untuk total tertentu, hingga menampilkan struk belanja.

## Fitur
- Daftar menu serta harga
- Penghitungan total belanja
- Diskon untuk total tertentu
- Struk belanja

## Teknologi
- Google Colab

## Cara Menjalankan
1. Buka Google Colab (https://colab.research.google.com)
2. Upload file program kasir (`.ipynb`)
3. Jalankan setiap cell dengan menekan tombol Run
